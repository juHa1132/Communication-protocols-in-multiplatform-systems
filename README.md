# Embedded Systems & Communication Protocols Projects

This repository contains a collection of projects written primarily in **C** and **Python**, along with some implementations in other languages where required by specific use cases.

The projects were developed and tested on a **Raspberry Pi 4B microcomputer** and were mainly focused on enabling and exploring communication between:
- Raspberry Pi and a personal computer
- Raspberry Pi and other microcontrollers such as **Arduino** and **STM32**

The implementations include various communication protocols and practical examples of data exchange in embedded and distributed systems.
